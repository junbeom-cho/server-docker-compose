# Codex AGENTS

## 프로젝트 목표

- 서비스별로 `docker-compose.yaml`을 정리하여 빠르게 배포할 수 있도록 한다.
- `docker-compose.yaml`별로 형식을 정형화하여 정리한다.

## Rule

### docker-compose.yaml format

- 참고 : `dockge/docker-compose.yaml`

```yaml
# ==================================================
# 서비스명 : 서비스 설명
# ==================================================

services:
  app:
    image: image/image:latest
    container_name: app
    restart: unless-stopped
    ports:
    volumes:
    environment:
      - key1=value1
      - key2=value2
# ==================================================
# 헬스체크
# ==================================================
    healthcheck:
# ==================================================
# 리소스 제한 - 필요시 조정
# ==================================================
    deploy:
# ==================================================
# 로그 설정
# ==================================================
    logging:
# ==================================================
# 네트워크 설정
# ==================================================
networks: {}
```

- 전체적인 형식은 위의 형식을 따르고 주석을 지킨다.
- 환경변수의 key:value가 어떤 기능을 하는지 간단하게 설명을 주석으로 작성한다.
